<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-lg dark:text-gray-400 text-gray-600 mb-8">Get Hiro CRM running in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the Hiro CRM repository and navigate to the frontend directory:
      
      ```bash
      git clone https://github.com/David-Cabeza/hiro-crm.git
      cd hiro-crm/frontend
      npm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Create a `.env.local` file with your Supabase credentials:
      
      ```bash
      cp .env.example .env.local
      ```
      
      Add your Supabase project URL and keys:
      
      ```bash
      NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
      NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
      SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
      ```
    </Step>
    
    <Step title="Set up the database">
      Run the database migrations using the Supabase CLI:
      
      ```bash
      supabase db push
      ```
      
      Or apply migrations manually via the Supabase dashboard SQL editor.
    </Step>
    
    <Step title="Start the development server">
      Launch the application:
      
      ```bash
      npm run dev
      ```
      
      Visit [http://localhost:3000](http://localhost:3000) to access your Hiro CRM instance.
    </Step>
  </Steps>
</div>
