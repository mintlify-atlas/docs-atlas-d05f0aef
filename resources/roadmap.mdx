## v1.0 — Initial Release (Current)

Launched March 2025 as an open-source project for the Spanish hospitality sector.

### CRM & Customers
- [x] Full customer profiles (visits, spending, history)
- [x] RFM segmentation (Recency, Frequency, Monetary)
- [x] Internal tags and notes per customer
- [x] Advanced search and filters

### Reservations & Operations
- [x] CoverManager integration (automatic sync)
- [x] Revo POS integration
- [x] Multi-location operational calendar
- [x] Event management and staff coordination

### Marketing
- [x] Email and SMS campaigns (Brevo)
- [x] Dynamic audience segments
- [x] Campaign metrics (open rate, click rate)
- [x] NPS and satisfaction surveys

### Loyalty Program
- [x] Configurable tiers (New → Founders)
- [x] Points system per visit and spending
- [x] Program management panel

### Analytics & AI
- [x] Executive dashboard with KPIs
- [x] AI Assistant (OpenAI GPT-4) — natural language queries
- [x] Cohort analysis and trends
- [x] Report exports

### Hotel CRM
- [x] Property and guest management
- [x] Integration with customer module

### Platform
- [x] Single-tenant (one deployment per business)
- [x] Authentication with Supabase
- [x] Multi-location support (restaurant groups)
- [x] Progressive Web App (PWA)
- [x] i18n (Spanish/English)
- [x] One-click deploy on Vercel

## v1.1 — Onboarding & UX (Q2 2025)

Improving the experience for new users installing the project.

- [ ] **Onboarding wizard** — guided setup on first login
- [ ] **Deploy to Vercel button** in README
- [ ] **Interactive demo** accessible without registration at `demo.hiro-crm.com`
- [ ] **Complete technical documentation** in `/docs`
- [ ] Accessibility improvements (a11y)
- [ ] E2E tests with Playwright for critical flows

<Note>
  **Status:** In development. Expected release in Q2 2025.
</Note>

---

## v1.2 — Communication & Automation (Q3 2025)

- [ ] **WhatsApp campaigns** (Brevo WhatsApp API)
- [ ] **Automatic workflows** — birthdays, win-back, post-visit
- [ ] **Customizable email templates** per brand
- [ ] Google Reviews integration
- [ ] Data export (GDPR compliance)
- [ ] Documented public API (OpenAPI spec)

---

## v1.3 — Customization (Q4 2025)

- [ ] **Configurable visual theme** (colors, logo, fonts) from the panel
- [ ] **Custom domain** for the admin panel
- [ ] Products/digital menu module
- [ ] Allergen management per dish
- [ ] QR codes for satisfaction surveys

---

## v2.0 — International Expansion (2026)

- [ ] **Complete multi-language** (PT, FR, IT, DE)
- [ ] Multi-currency support
- [ ] Integrations with other European POS systems (TheFork, OpenTable)
- [ ] Native reservation module (without CoverManager)
- [ ] **Mobile app** (React Native / Expo)
- [ ] Optional multi-tenant (for groups wanting SaaS)
